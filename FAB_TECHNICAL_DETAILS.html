Code Modules

  • UnrealMCPCore [Editor]: HTTP server, JSON-RPC 2.0 protocol, session management, 249 core tools, MCP Monitor Panel, settings UI, console commands. No optional plugin dependencies.
  • UnrealMCPCoreGAS [Editor]: 13 Gameplay Ability System tools. Requires GameplayAbilities plugin.
  • UnrealMCPCoreNiagara [Editor]: 11 Niagara VFX tools. Requires Niagara plugin.
  • UnrealMCPCorePCG [Editor]: 13 Procedural Content Generation tools. Requires PCG plugin.
  • UnrealMCPCoreStateTree [Editor]: 7 State Tree tools. Requires StateTree plugin.
  • UnrealMCPCoreSmartObject [Editor]: 5 Smart Object tools. Requires SmartObjects plugin.
  • UnrealMCPCoreEnhancedInput [Editor]: 7 Enhanced Input tools. Requires EnhancedInput plugin.
  • UnrealMCPCoreMetaSound [Editor]: 19 Audio/MetaSound tools. Requires Metasound plugin.
  • UnrealMCPCoreControlRig [Editor]: 6 Control Rig tools. Requires ControlRig plugin.
  • UnrealMCPCoreMotionDesign [Editor]: 5 Motion Design tools. Requires ClonerEffector plugin.

Extension modules are loaded dynamically at startup. If a required plugin is not installed, the extension module is silently skipped: all other tools remain available.

Number of Tools: 348 (249 core + 99 in extension modules)

Number of C++ Modules: 10 (1 core + 9 extensions)

Tool Categories: 30 (Blueprint, Asset, Level, Material, Animation, Widget/UI, DataTable/Struct, PCG, Niagara, Sequencer, Behavior Tree, Blackboard, State Tree, EQS, Smart Object, Mesh, Enhanced Input, GAS, Audio, Landscape, Physics, Foliage, World Partition, Control Rig, Rendering Config, Curve Asset, Motion Design, Project, PIE/Testing, Infrastructure)

Protocol: JSON-RPC 2.0 over MCP Streamable HTTP (+ SSE, + stdio proxy)

Transport: HTTP (POST /mcp) and stdio (included Python proxy, stdlib only)

External Dependencies: None: pure C++ plugin using only built-in UE modules

Network Replicated: No (editor-only plugin)

Thread Safety: All shared state protected by FCriticalSection locks. Multiple AI agents can connect simultaneously.

Supported Platforms

  • Development: Win64, Mac, Linux
  • Engine: Unreal Engine 5.5+

Integration Testing

2134 automated integration tests across all 348 tools. Python test runner (stdlib only) with parallel execution, auto-start headless editor, and per-tool performance benchmarks.
