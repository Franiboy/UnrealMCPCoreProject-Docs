v0.1-beta - Initial Release

Supported from Unreal Engine 5.5 onwards.

MCP Server & Protocol

  • HTTP server with JSON-RPC 2.0 over MCP Streamable HTTP transport.
  • Session management with auto-initialization: no handshake required, just send tool calls.
  • Multi-session support: multiple AI agents (Devin, Claude Desktop, Cursor, etc.) can connect simultaneously.
  • SSE (Server-Sent Events) response wrapping for MCP-compliant clients.
  • JSON-RPC batch request support.
  • stdio transport proxy (Python, stdlib only) for MCP clients that require subprocess communication.
  • CORS support, rate limiting (token-bucket per session), configurable tool timeout.
  • Full Project Settings UI (port, auto-start, CORS, verbose logging, rate limits).
  • Console commands: MCP.Start, MCP.Stop, MCP.Status.

MCP Monitor Panel

  • Status bar drawer with live server status, session count, tool count, and request stats.
  • Scrollable log table of all tool calls with details, summary, and clickable asset hyperlinks.
  • Revert (undo) and Retry actions from context menu with multi-select support.
  • Horizontal/vertical split layout, auto-scroll, dockable in any editor layout.
  • Client Config Helper popup with ready-to-paste JSON for HTTP and stdio transports.

348 Tools across 30 categories

  • Blueprint (41): full read/write access to variables, functions, macros, components, interfaces, event dispatchers, graph nodes, pin wiring, and class defaults.
  • Asset (14): list, inspect, rename, move, duplicate, delete, import/export, save.
  • Level / World (12): actors, transforms, world settings, streaming levels, viewport selection.
  • Material (17): create materials and instances, expression graph editing, parameter control.
  • Animation (28): AnimSequences, Montages, BlendSpaces, AnimBPs, skeleton sockets, virtual bones.
  • Widget / UI (23): create Widget Blueprints, build widget hierarchies, animations, event bindings.
  • DataTable / Struct (9): create and edit DataTables, UserDefinedStructs.
  • PCG (13): create and edit PCG graphs, execute/clear generation.
  • Niagara (11): systems, emitters, parameters, renderers.
  • Sequencer (13): Level Sequences, bindings, tracks, keyframes, playback.
  • Behavior Tree (7), Blackboard (6), State Tree (7), EQS (5), Smart Object (5): full AI tool chains.
  • Mesh (12), Enhanced Input (7), GAS (13), Audio (19), Landscape (9), Physics (12), Foliage (7).
  • World Partition (7), Control Rig (6), Rendering Config (6), Curve Asset (6), Motion Design (5).
  • Project (17): project metadata, class hierarchy, enum/struct introspection, console commands, log retrieval.
  • PIE / Testing (10): Play In Editor control, input injection, screenshots, automation tests.
  • Infrastructure (1): request log query for AI agents.

Extension Module Architecture

  • 9 optional-dependency tool categories isolated into separate DLL modules (GAS, Niagara, PCG, StateTree, SmartObject, EnhancedInput, MetaSound, ControlRig, MotionDesign).
  • Main plugin DLL has zero optional plugin dependencies: always loads, even on Fab projects without all plugins installed.
  • Missing extension DLLs are silently skipped at startup.

Infrastructure

  • Integration test framework (Python, stdlib only) with 2134 tests across all tools.
  • Parallel test runner with per-file sessions, auto-start headless editor, keep-alive connections.
  • Performance benchmarks with per-tool server-side timing stats and JSON export.
  • Fab packaging script (per engine version, auto-detects Full/Free edition).
