Give AI agents full access to your Unreal Editor. UnrealMCPCore is a pure C++ plugin that runs an MCP server inside the editor: connect any AI client (Claude Desktop, Devin, Cursor, or your own) and let it read, create, and modify Blueprints, Materials, Levels, Animations, and 25+ more asset categories. No Python or Node.js runtime required.

Free demo available on Fab: try the full plugin with a limited tool set.

How It Works

  1. Install the plugin: the MCP server starts automatically when the editor opens.
  2. Connect your AI client to http://localhost:8090/mcp (HTTP) or use the included stdio proxy for clients like Claude Desktop.
  3. Build: your AI agent can now create Blueprints, edit materials, spawn actors, wire nodes, configure animations, and much more. 348 tools at its disposal.

Why UnrealMCPCore?

  • 348 Tools, 30 Categories: From Blueprints and Materials to Niagara VFX, PCG, Sequencer, GAS, and Animation Blueprints: the most comprehensive MCP bridge for Unreal Engine available.
  • Zero External Dependencies: Pure C++ plugin. No Python runtime, no Node.js server, no Docker containers. Install, open the editor, connect.
  • Works With Any MCP Client: Supports HTTP (Streamable HTTP + SSE) and stdio transports out of the box. Claude Desktop, Devin, Cursor, or any JSON-RPC 2.0 client.
  • Multi-Agent Ready: Multiple AI agents can connect simultaneously, each with their own session. Thread-safe by design.
  • Fab-Safe Architecture: Optional engine plugins (GAS, Niagara, PCG, etc.) are isolated into separate extension DLLs. If a plugin isn't installed in your project, those tools are silently skipped: everything else works perfectly.
  • Built-In Monitoring: The MCP Monitor Panel shows every tool call in real time with details, revert (undo), and retry. Dock it anywhere in the editor layout.

What Can AI Agents Do?

  • Blueprints: Create classes, add variables/functions/components, wire graph nodes, set defaults, compile: full Blueprint authoring from AI.
  • Materials: Create materials and instances, build expression graphs, set parameters, compile.
  • Levels: Spawn actors, set transforms and properties, manage streaming levels, query world settings.
  • Animations: Create Montages, BlendSpaces, and AnimBPs. Add states, transitions, notifies, curves.
  • Widget UI: Build UMG interfaces programmatically: widget hierarchies, animations, event bindings.
  • VFX & Procedural: Niagara systems/emitters, PCG graphs, landscape sculpting, foliage placement.
  • AI Systems: Behavior Trees, Blackboards, State Trees, EQS queries, Smart Objects, GAS abilities/effects.
  • Sequencer: Level Sequences with bindings, tracks, keyframes, and playback control.
  • Project Management: Asset operations (import, export, rename, delete), class introspection, console commands, log retrieval.

MCP Monitor Panel

  • Live status bar showing server state, session count, and request stats.
  • Scrollable log of every tool call with clickable asset hyperlinks.
  • Right-click to Revert (undo) or Retry any operation.
  • Client Config Helper: one-click copy of the JSON config snippet for your MCP client.
  • Dockable in any editor tab: Level Editor, Blueprint Editor, Material Editor.

Early Access (v0.1-beta)

UnrealMCPCore is currently in version 0.1-beta. With 348 tools across 30 categories, the feature scope is enormous. Every tool has been integration-tested (2134 automated tests), but edge cases in your specific project setup may still surface.

If you encounter any issues, please don't hesitate to contact me directly. The more details you can provide, the faster I can help:

The same goes for feature requests: whether it's missing functionality in existing tools, new tools you'd like to see, or entirely new categories. I'm actively developing this plugin and your feedback directly shapes the roadmap.

Code Modules
